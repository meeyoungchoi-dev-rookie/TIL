## 문제

배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

## 입력

첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

## 출력

첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

## 예제 입력 1

```
2143

```

## 예제 출력 1

```
4321

```

## 예제 입력 2

```
999998999

```

## 예제 출력 2

```
999999998

```

## 예제 입력 3

```
61423

```

## 예제 출력 3

```
64321

```

## 예제 입력 4

```
500613009

```

## 예제 출력 4

```
965310000
```

### 무엇

- 숫자 N을 입력받는다
- N을 내림차순으로 정렬한다

### 내가 생각한 풀이 과정

- 숫자 N을 String 타입으로 입력받는다
- n을 char 타입의 배열로 만든다
- 배열을 오름차순으로 정렬한다 (Arrays 클래스의 sort 메서드 사용)
- 배열의 마지막 인덱스 부터 순차적으로 출력한다
- 왜? 내림차순으로 정렬해야 하기 때문

```java

public class 소트인사이드 {
    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);
        String n = scanner.nextLine();

        char[] array = n.toCharArray();
        Arrays.sort(array);

        for (int i = array.length - 1; i >= 0; i--) {
            System.out.print(array[i]);
        }
    }
}
```
